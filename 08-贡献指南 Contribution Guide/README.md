# 08-贡献指南 | Contribution Guide

> 每一个声音都重要，每一份贡献都有价值。

## 欢迎贡献

DevInAIWorld 是一个开放的社区项目，需要大家的共同参与。

## 如何贡献

### 方式一：提交内容
- 添加新的文章或指南
- 补充现有内容
- 翻译成其他语言
- 修正错误或过时的信息

### 方式二：分享案例
- 你的转型经历
- 你的学习心得
- 你的失业应对经验
- 你的创业故事

### 方式三：数据贡献
- 分享行业数据
- 提供薪资信息
- 分析招聘趋势

### 方式四：社区建设
- 参与讨论
- 帮助回答问题
- 传播项目理念

## 贡献流程

1. **提Issue** - 先开一个 Issue 说明你想做什么
2. **讨论方向** - 社区成员讨论可行性
3. **提交PR** - 按照模板提交 Pull Request
4. **社区审核** - 维护者进行审核
5. **合并发布** - 审核通过后合并

## 内容规范

- 理性、客观、不制造焦虑
- 事实为主，数据支撑
- 尊重不同观点
- 禁止人身攻击
- 不提供具体投资建议

## 联系方式

- GitHub Issues
- 提交 PR
