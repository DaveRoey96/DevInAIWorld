# 数据工程与AI工程

> 数据是 AI 的基础，数据工程师是 AI 时代的重要角色。

## 数据工程

### 是什么

构建数据管道，把数据从 A 传到 B，并处理好

### 核心工作

1. **数据采集**
   - 日志收集
   - 第三方 API
   - 数据库同步

2. **数据处理**
   - ETL
   - 数据清洗
   - 数据转换

3. **数据存储**
   - 数据仓库
   - 数据湖
   - 实时存储

4. **数据服务**
   - API
   - BI 支持
   - 分析支持

---

### 技术栈

| 层级 | 技术 |
|-----|-----|
| 编程 | Python, SQL, Scala |
| 大数据 | Spark, Flink, Kafka |
| 存储 | PostgreSQL, MySQL, MongoDB |
| 数仓 | Snowflake, BigQuery, Redshift |
| 湖 | Delta Lake, Iceberg, Hudi |
| 编排 | Airflow, Dagster |

---

### 技能要求

**必须**：
- SQL（必须精通）
- Python
- ETL 流程
- 数据建模

**加分**：
- 大数据技术
- 实时处理
- 云平台

---

|-----|---------|
| Junior Data Engineer | 15-25K |
| Data Engineer | 20-40K |
| Senior Data Engineer | 35-55K |

## AI 工程

### 是什么

把 AI 模型落地到产品中

### 核心工作

1. **模型开发**
   - 数据准备
   - 模型训练
   - 模型评估

2. **模型部署**
   - 模型服务化
   - A/B 测试
   - 监控

3. **MLOps**
   - 自动化流水线
   - 版本管理
   - 再训练

4. **AI 应用**
   - Prompt 工程
   - AI 集成
   - 效果优化

---

### 技术栈

| 层级 | 技术 |
|-----|-----|
| 编程 | Python（必须） |
| ML | PyTorch, TensorFlow, HuggingFace |
| 部署 | Docker, K8s, Triton |
| 向量 | Pinecone, Milvus, Qdrant |
| MLOps | MLflow, Kubeflow, Airflow |

---

### 技能要求

**必须**：
- Python（精通）
- 机器学习基础
- 模型部署
- API 开发

**加分**：
- 大模型微调
- Prompt 工程
- 分布式训练

---

### 岗位与薪资

| 岗位 | 薪资范围 |
|-----|---------|
| Junior AI Engineer | 20-30K |
| AI Engineer | 25-45K |
| Senior AI Engineer | 40-60K |
| ML Platform Engineer | 50K+ |

---

## 数据工程 vs AI 工程

### 数据工程

- 数据在先
- 稳定
- 需求持续

### AI 工程

- 模型在先
- 快速发展
- 需求爆发

---

## 如何选择

| 你更适合 | 选择 |
|---------|-----|
| 喜欢数据、SQL | 数据工程 |
| 喜欢 AI、模型 | AI 工程 |
| 想要稳定 | 数据工程 |
| 想要高薪 | AI 工程 |
| 想快速入行 | 数据工程 |

---

## 学习路径

### 数据工程

```
SQL → Python → ETL → Airflow → Spark → 云平台
```

### AI 工程

```
Python → ML 基础 → PyTorch → 模型部署 → 大模型 → MLOps
```
