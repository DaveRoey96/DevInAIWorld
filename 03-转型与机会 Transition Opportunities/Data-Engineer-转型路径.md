# Data Engineer 转型路径

> 数据是资产，Data Engineer 是炼金术士。

## 什么是 Data Engineer

### 核心职责

构建和维护数据管道

- 数据采集
- 数据处理
- 数据存储
- 数据服务

---

## 与相关岗位的关系

| 岗位 | 关注点 |
|-----|-------|
| Data Engineer | 数据管道 |
| Data Analyst | 数据分析 |
| Data Scientist | 数据模型/AI |
| BI Engineer | 报表/可视化 |

---

## 技能要求

### 核心技能

**必须**：
- SQL（精通）
- Python
- ETL 流程

**推荐**：
- 大数据技术
- 云平台
- 数据建模

---

### 技术栈

| 类别 | 技术 |
|-----|-----|
| 编程 | Python, Scala |
| 数据库 | PostgreSQL, MySQL |
| 大数据 | Spark, Kafka, Flink |
| 数仓 | Snowflake, BigQuery, Redshift |
| 编排 | Airflow, Dagster |
| 云 | AWS, GCP, Azure |

---

## 转型路径

### 从传统开发转型

```
SQL → Python → ETL → Airflow → Spark → 云平台
```

### 从数据分析转型

```
SQL 进阶 → Python → ETL → Airflow → 大数据
```
