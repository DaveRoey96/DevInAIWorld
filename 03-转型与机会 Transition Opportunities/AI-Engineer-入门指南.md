# AI Engineer 入门指南

> AI 时代最热门的高薪岗位。

## 什么是 AI Engineer

### 职责

把 AI 能力落地到产品中

- 微调大模型
- 构建 AI Pipeline
- 集成 AI API
- 优化模型性能

---

## 与其他 AI 相关岗位的区别

| 岗位 | 核心工作 | 重点 |
|-----|---------|-----|
| AI Engineer | AI 产品落地 | 工程能力 |
| ML Engineer | 模型训练优化 | 算法能力 |
| Data Engineer | 数据管道 | 数据能力 |
| Prompt Engineer | Prompt 优化 | 提示词能力 |

---

## 技能要求

### 必须技能

**编程**：
- Python（精通）
- 至少一门其他语言

**AI/ML**：
- 机器学习基础
- 深度学习概念
- 大模型使用

**工程**：
- API 开发
- Docker
- 云服务

---

### 加分技能

- 大模型微调
- LangChain/LlamaIndex
- 向量数据库
- MLOps

---

## 学习路径

### 阶段 1: 基础（1-2个月）

```
Python → 机器学习基础 → 深度学习基础 → OpenAI API
```

### 阶段 2: 进阶（2-3个月）

```
LangChain → 向量数据库 → RAG → Agent
```

### 阶段 3: 实践（持续）

```
项目实战 → 开源贡献 → 找工作
```

## 转型建议

### 从传统开发转型

1. 学 Python（如果不会）
2. 学 AI 基础概念
3. 做 AI 项目
4. 投简历

### 转型难度

- ⭐⭐⭐（中等）
- 有编程基础就不难
- 主要是思维转换

---

## 行动建议

### 立即开始

1. 学 Python
2. 用 OpenAI API 做个东西
3. 学 LangChain
4. 做项目
