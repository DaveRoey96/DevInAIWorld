# Platform/Cloud Engineer 方向

> 构建平台，让开发者更高效。

## 什么是 Platform Engineer

### 核心职责

构建和维护开发者平台

- CI/CD 流水线
- 容器化平台
- 监控告警
- 内部工具

---

## 与 DevOps/SRE 的区别

| 岗位 | 重点 |
|-----|-----|
| DevOps | 流程/自动化 |
| SRE | 可靠性/可用性 |
| Platform Engineer | 开发者体验/平台 |

---

## 技能要求

## 适合人群

### 适合

- 喜欢基础设施
- 喜欢自动化
- 喜欢帮助他人

### 不适合

- 只喜欢业务开发
- 不喜欢运维

---

## 转型路径

### 从运维/DevOps 转型

```
Docker → K8s → Platform → 开发者平台
```

### 从开发转型

```
学容器 → 学 K8s → 学平台化 → 做平台
```

## 未来趋势

### AI 对岗位的影响

- 自动化程度提高
- 但平台需求增加
- 需要支持 AI 工作负载
