## 关于项目

**DevInAIWorld —— AI时代开发者生存指南**

> 当AI浪潮来袭，每一位开发者都在思考：
> 明天，我还能继续写代码吗？35岁后，我该何去何从？我会被AI彻底替代吗？我还有未来吗？

**带着这些疑问，我们建立了开源项目DevInAIWorld**

**这是一份生存指南**

**不是理论，是行动。**

**不是一个人在做，是一群人在一起。**

**在浪潮之中，不让任何一个认真生活的人被抛下！**

---

### 这是一个集体智慧的项目

一个人的力量或许有限，但**一群人的智慧可以穿越周期**。

- 你可以把你的经验写下来，帮助其他开发者
- 你可以把你的故事分享出来，让更多人看到方向
- 你可以贡献数据，让分析更有说服力
- 你可以指出错误，让内容更准确

**每一位开发者的经验，都是宝贵的财富。**

**每一位开发者的贡献，都能帮助到其他人。**

这就是我们 —— **一个由开发者共建、共享、共享的生存指南**。

---

### 为什么做这个项目

我们看到开发者面临的四大困境：

| 困境 | 描述 |
|------|------|
| 📚 信息过载 | 每天无数文章讲AI趋势，但很少有能指导行动的内容 |
| 🧭 方向迷失 | 技术更新太快，不知道学什么；岗位变化太大，不知道往哪走 |
| 😰 心理焦虑 | 担心被AI替代，害怕35岁危机 |
| 🛠️ 行动匮乏 | 知道该改变，但不知道从哪里开始 |

所以我们做了这个项目 —— **一个理性的、可执行的、汇聚集体智慧的开发者生存指南**。

---

## 如何参与 🤝

我们欢迎所有理性贡献！

| 类型 | 描述 |
|------|------|
| 📝 提交文章 | 研究文章、技术分析 |
| 📖 分享故事 | 真实转型经历 |
| 📊 提供数据 | 行业数据分析 |
| 🔧 改进内容 | 优化翻译、修正错误 |

**贡献流程**

```
1. Open an Issue  →  说明想法
2. Discuss        →  社区讨论
3. Submit PR      →  提交修改
4. Review         →  社区审核
5. Merge          →  合并发布
```
